
  # Telemetry Data Dashboard

  This is the frontend for my project VeriOps: An Agent Observability and Monitorability Framework.

## Frontend env vars

- `NEXT_PUBLIC_API_BASE_URL` (or `VITE_API_BASE_URL`): Base URL for the VeriOps API.
- `NEXT_PUBLIC_VERIOPS_API_KEY` (or `VITE_VERIOPS_API_KEY`): API key header value. Defaults to `dev-key` in dev.
- `NEXT_PUBLIC_ENV` (or `VITE_ENV`): Optional environment label shown in the UI.

## How to run

- `npm i`
- `npm run dev`
